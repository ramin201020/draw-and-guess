# Fonts Directory

Place your custom font files here:

- YourFont.woff2 (recommended - best compression)
- YourFont.woff (fallback)
- YourFont.ttf (fallback)

Supported formats:
- .woff2 (best)
- .woff (good)
- .ttf (fallback)
- .otf (fallback)

After adding your font files, the CSS will automatically load them.
